Tic-Tac-Toe

A simple 2-player Tic-Tac-Toe game in which users can take turn entering a number from 1-9 to choose where their X or O should be placed. The game also has an additional feature allowing the players to increase the size of the board. For example, instead of a normal 3x3 board, players can play the game on a 5x5 board. However, the condition of 3 in a row still applies to win the game.

The game was developed in Eclipse using Java.
